Old 3DS & 2DS Guide

You must follow these steps exactly.

Only proceed with this guide if your system looks like either of the following pictures.

Guide

  1. Copy all the files to the SD Card.
  2. Backup your files before proceeding.
  3. Open the music app and run Soundhax for the Old 3DS (even if you have a 2DS).
  4. It will now downgrade your system. Be patient, it can take up to 10 minutes.
  5. Run Safehax and your system will boot into custom firmware.
